BINCOM Python Internship Solution

Instructions for Assessors
1. Ensure MySQL is running and the `bincom_test` database is loaded.
2. Run the Python file `BINCOM_Python_Internship.py` using Python 3.8+.
3. Follow the menu:
   - 1: Display Polling Unit results
   - 2: Display summed LGA results
   - 3: Insert new Polling Unit results
4. For inserting, the program automatically fills required fields (`entered_by_user`, `date_entered`, `user_ip_address`).

Note
- LGA totals use valid LGA IDs.
